High-performance compute kernels, operators, and tensor APIs for MAX graphs.

The MAX AI kernels library provides the building blocks for AI inference and
other compute-intensive workloads on CPU and GPU, including
[linear algebra](/max/api/kernels/linalg/) functions like matrix
multiplication, [neural network operators](/max/api/kernels/nn/) such as
attention and convolution, [quantization](/max/api/kernels/quantization/)
routines, [key-value caches](/max/api/kernels/kv_cache/) for transformer
models, and primitives for
[multi-GPU communication](/max/api/kernels/comm/) and
[extending a MAX graph](/max/api/kernels/extensibility/) with
custom operations.

</section>
