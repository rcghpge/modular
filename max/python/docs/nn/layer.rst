Base classes for building graph-based neural network layers.

The :class:`~max.nn.layer.LayerList` class provides an ordered container for
composing multiple layers. The :class:`~max.nn.layer.Layer` base class is
deprecated — use :class:`~max.experimental.nn.module.Module` for new code instead.
