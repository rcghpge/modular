diagnostics
-----------

Diagnostic utilities for monitoring and analyzing MAX platform performance.

The ``max.diagnostics`` package provides tools for inspecting system behavior
and hardware resource usage during model inference. Use these utilities to
profile GPU utilization and memory consumption for performance tuning.

* :code_link:`/max/api/python/diagnostics/gpu|gpu`: Real-time GPU monitoring and diagnostic tools for NVIDIA and AMD hardware.
