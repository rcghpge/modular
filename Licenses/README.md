# Overview

This directory contains the license files that should be included in
all packages that are shipped externally as part of Modular products.

The licenses may be different per product - for example, the Modular
AI Engine, Mojo (SDK) have different licenses (and in some cases, we
may have free versions of the AI Engine for non-production use).
Thus, the licenses are organized by directory.

The `Third-Party-Notices` file is a common attribution of all 3rd party
software (both OSS and proprietary) and their appropriate licenses used
in the products.
