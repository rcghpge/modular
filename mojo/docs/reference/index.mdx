Welcome to the Mojo language reference. This documentation provides a concise
reference for Mojo's syntax—identifiers, keywords, literals, expressions,
operators, statements, function declarations, and more. It's organized by
grammar construct and designed for quick lookup when you need the exact
rules, precedence, or form of a specific language element.
